# 微信对话生成--Qt版

## 简介

​	本软件是一款微信聊天记录生成的的工具，基本使用操作很简单，右键点击某个消息即可删除消息，双击则可以转换发送者，点击双方头像可以设置头像，双击点击微信名即可编辑微信名，支持图片发送，支持emoji表情发送，支持一键演绎发送过程。

## 功能列表

| 功能项                                                       | 是否支持 |
| ------------------------------------------------------------ | -------- |
| 图片消息模拟、emoji表情消息模拟、系统消息模拟                | ✅        |
| 头像、背景、微信名设置                                       | ✅        |
| 自动重新发送演绎                                             | ✅        |
| 语音消息模拟、红包消息模拟、转账消息模拟、未读消息数、定位消息 | 后期支持 |
| 不同分辨率适配                                               | 后期支持 |
| 聊天信息列表页面                                             | 后期支持 |
| 朋友圈页面                                                   | 后期支持 |
| 微信钱包页面                                                 | 后期支持 |



## 依赖

- Qt5.xx (及以上)
- C++11(及以上）
